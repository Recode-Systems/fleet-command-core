# Fleet Management System for Kenya Electricity Generating Company

## Overview

The Fleet Management System is a web-based application developed to improve the monitoring, tracking and management of fleet operations within the Kenya Electricity Generating Company (KenGen).

The system was developed in response to operational challenges observed within existing fleet management processes, including misuse of company vehicles, lack of accountability in repairs and inefficient tracking of fuel and maintenance resources.

The project provides a centralized platform for monitoring vehicle activity, managing maintenance records and improving operational transparency through digital tools and real-time visualization.

---

## Problem Addressed

The existing fleet management processes relied heavily on manual systems such as:
- Physical logbooks
- Spreadsheet tracking
- Manual reporting
- Word document based submissions

This created several challenges, including:
- Difficulty locating vehicles
- Misuse of company vehicles
- Poor accountability in repair processes
- Inefficient fuel tracking
- Delays in documentation and reporting

The system was designed to address these challenges through centralized digital fleet management.

## System Architecture

The system follows a component-based architecture where:
- React handles UI rendering and state management
- React Leaflet handles geospatial visualization
- Modular components manage dashboards, forms and tracking interfaces
